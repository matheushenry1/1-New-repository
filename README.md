# Site Gás Network Engenharia

Versão estática pronta para GitHub Pages, com layout premium, páginas institucionais e 50 páginas individuais de serviços.

## Estrutura
- `index.html`
- `sobre.html`
- `contato.html`
- `servicos.html`
- `servicos/*.html`
- `assets/css/styles.css`
- `assets/js/app.js`
- `assets/images/*.svg`
- `robots.txt`
- `sitemap.xml`

## Publicação
1. Envie todos os arquivos para a raiz do repositório.
2. No GitHub, abra **Settings > Pages**.
3. Em **Deploy from a branch**, escolha a branch `main` ou `principal` e a pasta `/ (root)`.
4. Aguarde a publicação.

## Observações
- Os links internos foram configurados com caminhos relativos para funcionar no GitHub Pages.
- O site usa SVGs locais como imagens de apoio.
- Quando o logo oficial estiver disponível, ele pode substituir a marca textual facilmente.
- O domínio de referência configurado em canonical e sitemap é `https://www.gasnetwork.org`.
